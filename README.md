# Gym Program Builder

## Project Overview

This project is a web-based Gym Program Builder designed to help coaches and individuals create structured 7-day workout programs in a fast and efficient way.

The main goal of this project is to replace traditional tools such as Excel or Microsoft Word, which are slow and not optimized for building and managing training programs. This application focuses on speed, simplicity, and a smooth user experience.

## Purpose

The reason behind building this project is to simplify the process of creating and managing gym programs for both personal use and client coaching.

Instead of manually formatting workout plans in spreadsheets or documents, this tool provides a structured and interactive interface where programs can be built, edited, and exported easily.

## Features

- Create a 7-day workout program
- Add multiple exercises per day
- Define sets, reps, RIR, and optional notes for each exercise
- Edit and update programs dynamically
- Save programs for future editing
- Export the full program as a PDF

## Technology Used

The project is built using:

- HTML for structure
- CSS for styling
- Vanilla JavaScript for functionality and interactivity
- PHP for backend API handling
- PostgreSQL for database storage

No frontend frameworks or libraries were used to keep the project lightweight and focused on performance and simplicity.

## How It Works

1. The user creates a new workout program.
2. Each program contains 7 days, and each day contains multiple exercises.
3. The user fills in exercise details such as sets, reps, and RIR.
4. The data is saved into a PostgreSQL database through a PHP API.
5. The user can later retrieve and edit any saved program.
6. The final program can be exported as a clean PDF file for printing or client delivery.

## Design Philosophy

The project is built with a strong focus on:
- Fast interaction speed
- Minimal and clean user interface
- Easy program creation without unnecessary steps
- Practical use in real coaching environments

## Future Improvements

- User authentication system for multiple coaches

## Conclusion

This project was developed as a practical tool to improve the workflow of creating gym programs. It aims to provide a faster and more efficient alternative to traditional document-based methods while maintaining simplicity and usability.
